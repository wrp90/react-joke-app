
# Project Title

This is a simple CRUD application that searches for a joke from the jokeAPI. The user can save the joke, delete or post it to twitter.


## Technologies Used

* JavaScript
* React
* Redux
* React Bootstrap
* bcrypt
* Sequelize
* PostgreSQL

## Run Locally

Clone the project

```bash
  git clone git@github.com:wrp90/react-joke-app.git
```

Go to the project directory

```bash
  cd react-joke-app
```

Install dependencies

```bash
  npm install
  cd client
  npm install
  cd..
  cd server
  npm install
  cd..
```

Start the app

```bash
  npm run dev
```
